# Environment variables

Required:

- `SUPABASE_URL`
- `SUPABASE_KEY` (or `SUPABASE_SERVICE_ROLE_KEY`)
- `ADMIN_PASS`
- `guild`
- `DISCORD_URL`

Optional:

- `PORT`
- `HYPIXEL_API_KEY` (or `SECRET_HYPIXEL_API_KEY`)
- `SUPABASE_STORAGE_BUCKET`
- `SUPABASE_STORAGE_OBJECT`
- `SUPABASE_TABLE`
- `PLACEMENT`
- `PLACEMENT_TOKEN`
